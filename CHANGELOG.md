## [2.0.0] - 2026-07-11

### Added
- Repository scaffold with 29 category directories
- Standardized JSON schema with 18 required fields
- 77 high-quality project entries across all categories
- Python automation scripts:
  - `validate.py` — JSON schema validation
  - `build_index.py` — Search index generation
  - `health_check.py` — URL and repository health checks
  - `generate_report.py` — Validation reports
  - `ai_categorize.py` — AI-powered categorization
  - `ai_assistant.py` — Interactive tech recommendations
  - `ai_knowledge.py` — Summaries, comparisons, use cases
  - `search_engine.py` — Fuzzy search with relevance ranking
  - `relationships.py` — Project relationship graph
  - `recommendations.py` — Smart recommendation engine
  - `scoring.py` — Quality scoring (0-10)
  - `analytics.py` — Dashboard and monitoring
  - `cache.py` — TTL-based caching system
- REST API (FastAPI) with 20+ endpoints
- Modern responsive web interface (dark theme)
- 7 GitHub Actions workflows for automation
- Full documentation (9 docs)
- Community templates (issues, PR, Code of Conduct)
